  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    Java es un lenguaje compilado mientras que JavaScript es un lenguaje interpretado, además Java está orientado a objetos mientras que JavaScript está orientado a prototipos

  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    • getFullYear: una vez que se ha creado una instancia este método devuelve el año completo de la fecha.

    • getMonth: este método retorna el mes de la fecha creada

    • toString: nos devuelve la fecha y hora en forma de un string

    • getTime: devuelve los milisegundos transcurridos desde 01/01/1970

    • getDay: metodo que devuelve el día de la semana de la fecha

  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    • map: método que se utiliza para iterar el arreglo agregando una funcion como argumento que recibe como argumentos el elemento, su índice y el arreglo completo por cada iteración y retorna un nuevo arreglo de los valores que se retornen dentro de la función.

    • forEach: método que se utiliza para iterar el arreglo que recibe una función al igual que el método map pero que no retorna ningún valor, este método sólo itera el arreglo.

    • filter: método que se utiliza para filtrar un arreglo, al igual que los dos métodos anteriores recibe una función como argumento que tiene como parametros los mismos que las dos funciones anteriores, este método retorna un arreglo con los elementos para los que la función haya retornado true

    • sort: método que sirve para ordenar un arreglo, este ordenamiento depende del unicode, sin embargo, si se desea hacer un ordenamiento específico se puede añadir una función de comparación que tiene dos parametros a y b que son los elementos del array que se van a comprarar y dependiendo de si esta devuelve un valor negativo (a menor b) cero (a igual b) o positivo (a mayor b) el algoritmo de ordenamiento interpretará cómo reordenar el arreglo. Este método afecta al algoritmo original.

    • reduce: método igual a map con la diferencia de que además de una funcion recibe un numero que será el valor inicial de un contador que será pasado a la funcion como primer argumento en cada iteraci´n y será modificado hasta que la función reduce lo retorne al final.

  • ¿Cómo se declara una variable con alcance local dentro de una función?

    Una variable con alcance local del scope de una función se debe declarar dentro de la función y esta se puede declarar de tres formas que siguen la siguiente estructura: var hola = "hola Mundo".
    Además de var, se puede usar let y const en vez de var con la diferencia de que estas son block scoped, es decir, si se declaran dentro de un if estas no serán accesibles fuera de este if cuando var si lo sería.

  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Que podríamos afectar partes del código que hacen uso de estas variables y generar errores que serían difíciles de corregir porque en js no se puede elegir si las variables se pasan como referencia o como valor a las funciones, esto dependerá del tipo de dato del valor.
